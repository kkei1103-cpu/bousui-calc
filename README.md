# ウレタン防水 缶数計算ツール（田島 GO-JIN／オルタックシリーズ）

田島ルーフィングのウレタン塗膜防水材の **必要缶数を平場（㎡）／立上り（㎡）／側溝（m）から自動計算** する単体HTMLツールです。サイト全体がこの1ツールで構成されています。

## 使い方

`docs/index.html` をブラウザで開くだけ（ビルド不要・オフライン可）。スマホでもPCでも動きます。

1. **製品シリーズと工法** を選ぶ（オルタックエース／GO-JIN、密着・通気緩衝など）
2. **数量を入力** — 平場㎡／立上り㎡／側溝m（側溝は「長さ×展開幅」で面積換算）を必要な分だけ
3. **ロス率** を入れる（端部・ダレ・調合ロスの割増、初期5％）
4. **必要缶数** が材料ごと（プライマー・主材・トップコート…）と合計で自動表示

## 使用量（kg/㎡）について

- 初期値は **メーカー公開カタログを参考にした目安** です（例：オルタックエース平場 約2.0kg/㎡、GO-JIN 主材20kg/セット・補強布不要）。
- **工法・仕様書・下地状態で実際の数量は変わります。** 発注前に必ず該当工法の田島ルーフィング仕様書／積算資料で確認してください。
- 「使用量の設定」で係数（平場・立上り/側溝の kg/㎡、荷姿 kg/缶）を自由に編集でき、**ブラウザ（localStorage）に自動保存** されます。
- 「初期値に戻す」でいつでもリセットできます。

## 計算式

```
必要量(kg) = (平場㎡ × 平場kg/㎡ + (立上り㎡ + 側溝㎡) × 立上りkg/㎡) × (1 + ロス率)
側溝㎡    = 側溝長さ(m) × 展開幅(m)
必要缶数  = ceil(必要量 ÷ 荷姿kg/缶)
```

## 公開（GitHub Pages）

API・ビルド・サーバーは不要です。`docs/index.html` をブラウザで開けばそのまま使えます。

Webで公開する場合は **GitHub Pages** が簡単です：

1. このPRを `main` にマージ
2. リポジトリの **Settings → Pages** を開く
3. Source を「**Deploy from a branch**」、Branch を「**main**」／フォルダ「**/docs**」に設定して Save
4. 数分後、公開URL **https://kyu-999.github.io/bousui-calc/** で表示されます

`docs/` フォルダを任意の静的ホスティング（Netlify／社内サーバー等）に置いても動きます。
